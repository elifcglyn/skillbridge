# SkillBridge Project Plan

SkillBridge is organized as a full-stack application in a single repository. The frontend owns the browser experience, the backend owns API and database access, and docs capture planning decisions.

## Goals

- Keep the existing Vite + React frontend working.
- Add a Node.js + Express + TypeScript backend foundation.
- Move Prisma ownership into the backend.
- Keep real environment values out of Git.

## Current Milestones

- Establish full-stack folder structure.
- Document database, API, frontend, and task plans.
- Add health-check API foundation.
- Preserve the existing Supabase-powered frontend flow.
